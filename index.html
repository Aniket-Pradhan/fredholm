Solving Fredholm equations numerically.

Fredholm equations of the first kind (inhomogenious) can be written as follows:




  
  
g(y) = \int_a^{b} K(y,x) f(y) dx

Here functions K(y,x) (called Kernel) and g(y) (inhomogeneous term) are given and f(s) is the function to solve for.

In this form, such equation is often encountered in the quantum many-body physics. Depending on the context and details, they are referred to as either integral equations for quantum scattering problem, Dyson's equation or Bethe-Soppiter but have similar underlying feature.

For such applications, some common features arise:

  1. The delta function term could be singled out.

  2. If real and complex parts are separated, function diverges under the integral, principal value integral. There are resonances/bound state amplitute contributions.

    3. Taking this into account, we now get the following equation form: 
    

    
  
      
g(y) = f(y) + \textrm{P.v.} \int_a^{b} K(y,x) f(y) dx
    The Fredholm library is developed with such purposes in mind.

Usage.

My intention is to make it a package , however, there is a lot do be done first. So, chances are, working with the library can more challanging to date than I wish. Some basic knowledge of the python ecosystem is still a must unfortunately.

So ,if you are still not discouraged, let's get down to bussiness (to defeat the Huns!). First you need to make sure you have Python 2.7+ with the numpy packages properly installed at your system of choice. To see if that is the case try:

python import numpy

If there is no exception, the numpy probably works.

Now we can download the package and tun tests:

python ./fredholm/tests/test_main.py
If tests worked properly, we can now proceed to our first example case. Let us consider the following equation: 



  
  
y = f(y) + \textrm{P.v.} \int_0^{100} \frac{f(x) dx}{x^2 + y^2}
We import the package library and set up the given functions, and run the .solve() method. 
 
  #!/usr/bin/env python
  import Fredholm
  Fredholm.Kernel = lambda x,y: 1/(x**2 + y**2)
  Fredholm.f = lambda x: 1.
  Fredholm.range.low, Fredholm.range.high = 0. , 100.
  Fredholm.solve()
Horray! We got first soution. We can plot it or look up specific values: 
 
    Fredholm.solution(3.)
    Fredholm.plot()
That should get you starting.
